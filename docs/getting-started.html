创建页面

1. 下载/安装

First of all, you should download jQuery or Zepto. You can get zepto.js on http://zeptojs.com/, and jQuery on http://jquery.com/. It's recommand to use jQuery instead of Zepto.

pageui 依赖jQuery或者Zepto，你可以在 http://zeptojs.com/ 上下载Zepto, 或者在 http://jquery.com/ 下载jQuery。 推荐使用jQuery而不是Zepto。

Then, you should install pageui. It's recommand to install pageui with npm - npm install pageui.

然后需要下载 pageui. 推荐使用 npm 来安装 - npm install pageui.

或者在 github 上 clone 到本地并且编译一遍。如果你不想自己编译，可以在clone完之后选择 build 分支，里面有全部编译好的文件。或者更懒一点话，你可以直接点击 这里下载。

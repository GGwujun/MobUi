关于 pageui

pageui 是基于 Framework7 完成的一套轻量级UI库。如果你喜欢 Framework7，但是觉得它的兼容性不好或者太重量，那么 pageui 将是一个很好的选择。

{% include ad.html %}

兼容性

兼容到 iOS 6+ 以及 Android 4.0+

因为安卓机型众多，我们无法对每一台机器进行测试，如果你发现在安卓 4.0以上的机器上有bug，可以在 github 上给我们提 issue,或者直接联系我们（联系方式在下面）。

Changelog

v0.4.3 - 2016-03-28

  • 修复了 地址选择器中设置初始值的bug
  • 现在可以在一个页面有多个 地址选择器 和 时间选择器，他们不会相互影响
  • router 现在会忽略 href="javascript:;" 链接
  • Swiper 升级到 to V3.3.1
  • 修复了 Toast 不居中的问题
  • 修复了部分安卓设备中 calendar 和 picker 会弹出系统键盘的问题
  • 修复了 图片浏览器无法进行手势缩放的问题
  • 增加了一个demo：在tab中使用下拉刷新和无线滚动

v0.4.2 - 2016-03-14

这个版本主要是修复经常反馈最多的几个bug

  • 修复了picker上滑动会导致页面滚动的bug
  • 修复了 toast 不在页面居中的bug
  • 现在Modal和ActionSheet都可以通过点击背景来关闭，并且可以配置
  • 修复了路由器的bug: .back 链接上如果没有 href 属性会重定向到 undefined 页面.
  • 修复了路由器的bug: $.router.reloadPage() 会引起历史的问题，导致返回上一页出现bug.

v0.4.1 - 2016-01-28

  • 修复了路由器在加载内联页面时的JS报错

v0.4.0 - 2016-01-25

  • Update the doc.
  • Publish light7 on npm.
  • Remove unused file.

v0.3.1 - 2016-01-18

  • 修复了 Picker 宽度的bug

v0.3.0 - 2016-01-15

  • 新组件Notification.
  • 修复了 router 中的bug: 现在删除页面的时候，路由器会自动删除相关的panel和popover等。
  • 修复了 Fastclick 的一个bug: 对label的处理可能会导致stackoverflow。
  • $.fn.index 改为 $.fn.indexList, 因为jQuery本身就有一个方法叫 index. 参见 Contacts List.
  • 移除了hairline，使用border。因为hairline的实现在部分安卓机器上有性能问题，特别是滚动的时候可能会闪烁。

v0.2.1 - 2015-12-28

  • 为 Contacts List 增加了一个iOS风格的索引条
  • 修复了一些文档中 light7-extend.js 还没有改成 light7-swiper.min.js 的问题

v0.2.0 - 2015-12-24 - Merry Christmas!

  • 修复bug：cssmin 在压缩的时候会自动进行合并，会导致一些重要的兼容性代码被合并掉，在安卓机器上会出现问题。
  • 新组建 - Swipeout.
  • 支持 i18n - I18N.
  • 修复了 Side Panel 的bug: 如果跳转页面，会导致无法滑动打开侧栏。
  • 注意! light7-extend.js 更名为 light7-swiper.js, light7-extend.css 更名为 light7-swiper.css.
  • Modal 增加了一个新的 closePrevious 参数，在打开对话框的时候会先关闭之前的对话框.
  • Text 中增加了新的颜色和背景类， 同时也可以用在 Action Sheet 中.

v0.1.0 - 2015-12-20

  • 主仓库 Light7 全部翻译为英文. 中文仓库是独立的 Light-CN，这个中文仓库只进行文档翻译，不要用这里的代码.
  • Light7 同时兼容 jQuery 和 Zepto，你可以自由选择合适的库.
  • 增加了一个新的组件: Popover.
  • 修复bug: 在head中引入 light7.js 时，fastclick 和 modal 初始化的时候document.body为空，导致出错。
  • 在 router 配置中增加了一个 replace 选项，可以在加载新页面的时候在历史记录上替换掉当前页面. 同时增加了一个 reloadPage 可以刷新当前页面.
  • router 在加载新页面的时候 会自动替换掉相同id的旧页面.
  • 修复了fastclick不生效的bug：会导致手机上300ms的点击延迟

V0.0.5 - 2015-11-30

  • 修复了 路由 在safari中的bug，会导致safari在页面加载完成之后直接后退一次。
  • 修复了 日期时间选择器 没有格式化"小时"的问题：现在小时也会被格式化成两位数。
  • 修复了 侧栏 弹出后在safari中会导致页面横向滚动的问题。
  • 更新 picker，现在会自动读取 input 的值作为默认值。

V0.0.4

  • 修复了 幻灯片 中的一处代码错误，会导致幻灯片无法自动初始化。
  • 更新了 popup 文档中代码的错误: 如果想自动打开 popup，需要加上 open-popup 类才可以。
  • 更新了 路由 : 现在你可以使用路由进行工具栏切换了。修复了 pageReinit 的bug。 修复了当新页面只有一个 .page 容器时的错误。
  • 现在你可以显示和隐藏工具栏了。
  • 增加了一个新的 案例 页面。

V0.0.3 - 2015-11-03

  • 修复了使用 路由器时，如果刷新了页面，回退按钮需要点击两次才生效的bug
  • 修复了 图片浏览器文档demo中的错误
  • 更新了 幻灯片 的文档

V0.0.2 - 2015-10-29

  • 为 路由器 增加了一个 $.config.router 配置。现在你可以全局关闭路由器了。
  • 修复了一些文档中的错误。
